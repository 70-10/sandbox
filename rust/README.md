# Rust

## Installing

```cli
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Reference: [rustup](https://rustup.rs/)

### Settings Visual Studio Code

- Install [`rls-vscode`](https://github.com/rust-lang/rls-vscode) plugin.

## Create project

```cli
cargo new <PROJECT NAME>
```
